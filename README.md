# embedded-robotic-sorting-classifying-system

## Overview
This project is an automated sorting system that detects and classifies objects based on their material using sensors and a robotic gripper.

## Features
- Detects object presence using IR sensor
- Identifies metallic objects using metal sensor
- Sorts objects into two categories
- Automated conveyor system
- Real-time control using microcontroller

## System Architecture
The system consists of:
- Conveyor belt driven by DC motor
- Metal detection sensor
- IR sensor for object detection
- Servo motors controlling a robotic gripper
- Microcontroller for processing and control

## Technologies Used
- C/C++ (Arduino programming)
- Embedded Systems
- Sensors integration
- Basic automation logic

## How It Works
1. Object moves on conveyor belt  
2. IR sensor detects presence  
3. Metal sensor determines material  
4. Microcontroller processes signal  
5. Servo moves object to correct box
